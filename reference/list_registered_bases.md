# List Registered Parametric Basis Classes

List Registered Parametric Basis Classes

## Usage

``` r
list_registered_bases()
```

## Value

Character vector of registered class names.

## Examples

``` r
list_registered_bases()
#> [1] "BSpline"      "Ident"        "Poly"         "RobustScale"  "Scale"       
#> [6] "ScaleWithin"  "Standardized"
```
